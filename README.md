# Portfolio Website with React & Sanity 🚀

A modern and fully responsive developer portfolio website built using React.js, SCSS, Framer Motion, and Sanity.io as the headless CMS. This project showcases animations, dynamic content management, reusable components, and modern frontend best practices.

## ✨ Features

- Responsive Modern UI/UX
- Smooth Animations using Framer Motion
- Dynamic Content Management with Sanity
- Reusable React Components
- SCSS Styling Architecture
- Portfolio Projects Section
- Skills & Experience Section
- Testimonials Section
- Contact Form
- Fully Deployable Frontend & Backend

---

## 🛠️ Tech Stack

### Frontend
- React.js
- SCSS
- Framer Motion
- React Icons

### Backend / CMS
- Sanity.io

### Deployment
- Netlify / Vercel
- Sanity Studio Hosting
